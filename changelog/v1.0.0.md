# v1.0.0 (2026-06-18)

馬刺しMOD 初回リリース。Minecraft 1.20.1 / Forge・NeoForge 対応。

## 追加アイテム（食料4種）

| アイテム | 空腹度 | 隠し満腹度 |
|----------|:---:|:---:|
| 馬刺し (basashi) | 3 | 1.8 |
| 馬のタタキ (uma_no_tataki) | 8 | 12.8 |
| 馬肉のユッケ (uma_yukke) | 6 | 7.2 |
| 馬肉のタルタルステーキ (uma_tartare) | 11 | 19.8 |

- 全アイテムを「食料と飲み物」クリエイティブタブに追加。

## 入手・ドロップ

- 馬（普通の馬）を倒すと **馬刺し** がドロップ（1〜2個、ドロップ増加エンチャント対応）。
- **燃えている馬**（火属性で死亡／死亡時に炎上）を倒すと、馬刺しの代わりに **馬のタタキ** がドロップ。
- バニラの革ドロップはそのまま維持。

## レシピ

- **馬のタタキ**: 馬刺しを かまど / 燻製機 / 焚火 で焼く。
- **馬肉のユッケ**: 馬刺し ＋ 卵（作業台・並べ方自由）。
- **馬肉のタルタルステーキ**: 馬刺し ＋ ニンジン ＋ ビートルート ＋ 卵（作業台・並べ方自由）。

## 技術・対応環境

- Minecraft 1.20.1。
- **Forge / NeoForge 両対応**（1.20.1 では NeoForge が Forge 互換のため、Forgeビルド1つで両ローダーに対応）。
- Architectury（common + forge 構成）。
- 16×16 ドット絵テクスチャ。
- 多言語対応（日本語 / 英語）。
